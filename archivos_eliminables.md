# Archivos eliminables sugeridos

- `.firebase/logs/vsce-debug.log`: Archivo de log de depuración.
- `.idx/dev.nix`: Archivo de configuración de entorno de desarrollo (Nix), probablemente no esencial para producción.
- `.idx/icon.png`: Icono de entorno, no esencial para la app.
- `docs/blueprint.md`: Documentación interna, no afecta la lógica de la app.
- `mhda/`: Subproyecto o módulo que no parece estar integrado con el frontend principal (no hay referencias en el código principal).
